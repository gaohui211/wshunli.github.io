GCJ-02相关问题（在线服务）

发表于 2016-04-03   |   分类于 GCJ-02   |     |   本文阅读量 次

GCJ-02相关坐标转换的接口（在线服务）：

http://www.zdoz.net/apiList.html

支持世界标准地理坐标(WGS-84) 、中国国测局地理坐标（GCJ-02）<火星坐标> 、百度地理坐标（BD-09)之间互转

支持WGS转GCJ纠偏接口、GCJ反向转WGS接口
WGS转BD2纠偏接口、BD2反向转WGS接口
GCJ转BD2接口、BD2转GCJ接口
百度地图导航等

GCJ-02是由中国国家测绘局制订的地理信息系统的坐标系统。
它是一种对经纬度数据的加密算法，即加入随机的偏差。
国内出版的各种地图系统（包括电子形式），必须至少采用GCJ-02对地理位置进行首次加密。

批量纠偏接口(POST)

接口地址

http://api.zdoz.net/transmore.ashx

接口说明

批量纠偏，一次最大可纠偏1000个坐标点
参数
lats:维度，多个维度用“;”隔开
lngs:经度，多个经度用“;”隔开（要注意经纬度个数相等）
type:转换类型 【1.WGS -> GCJ】 【2.GCJ -> WGS】 【3.GCJ -> BD】 【4.BD -> GCJ】 【5.WGS -> BD】 【6.BD -> WGS】
返回值JSON
根据次序返回一个json格式的数组
演示
参数：

lats=34.123;34.332;55.231&lngs=113.123;112.213;115.321&type=1

返回：

[{"Lng":113.12942937312582,"Lat":34.121761850760855},{"Lng":112.21911710957568,"Lat":34.3306763095054}, {"Lng":115.33036232125529,"Lat":55.232930158541052}]

WGS转GCJ纠偏接口

接口地址

http://api.zdoz.net/transgps.aspx

接口说明

根据GPS经纬度换算出偏移后的火星坐标

参数
lat:纬度
lng:经度
演示地址
http://api.zdoz.net/transgps.aspx?lat=34.121&lng=115.21212

GCJ反向转WGS接口

接口地址

http://api.zdoz.net/gcj2wgs.aspx

接口说明

根据火星坐标得出纠偏前的WGS坐标
参数
lat:纬度
lng:经度
演示地址
http://api.zdoz.net/gcj2wgs.aspx?lat=34.121&lng=115.21212

WGS转BD2纠偏接口

接口地址

http://api.zdoz.net/transgpsbd.aspx

接口说明

根据GPS经纬度换算出偏移后的百度火星坐标
参数
lat:纬度
lng:经度
演示地址
http://api.zdoz.net/transgpsbd.aspx?lat=34.121&lng=115.21212

BD2反向转WGS接口

接口地址

http://api.zdoz.net/bd2wgs.aspx

接口说明

根据百度坐标获取WGS坐标
参数
lat:纬度
lng:经度
演示地址
http://api.zdoz.net/bd2wgs.aspx?lat=34.121&lng=115.21212

GCJ转BD2接口

接口地址

http://api.zdoz.net/gcj2bd.aspx

接口说明

根据Google地图坐标（GCJ-02）转为百度地图坐标（BD-09）
参数
lat:纬度
lng:经度
演示地址
http://api.zdoz.net/gcj2bd.aspx?lat=34.78301&lng=113.77581

BD2转GCJ接口

接口地址

http://api.zdoz.net/bd2gcj.aspx

接口说明

根据百度地图坐标（BD-09）转为 Google地图坐标（GCJ-02）
参数
lat:纬度
lng:经度
演示地址
http://api.zdoz.net/bd2gcj.aspx?lat=34.789281&lng=113.782224

根据地理位置获取

接口地址

http://api.zdoz.net/geo2loc.aspx

http://api.zdoz.net/geo2loc_2.aspx

接口说明

根据GPS经纬度得到地理位置
参数
lat:纬度
lng:经度
演示地址
type:GPS类型 【1.BD-09】 【2.GCJ-02】 【3.WGS-84】
演示地址
http://api.zdoz.net/geo2loc.aspx?lat=34.121&lng=115.21212&type=1

http://api.zdoz.net/geo2loc_2.aspx?lat=34.121&lng=115.21212&type=1

Google地图导航

接口地址

http://api.zdoz.net/daohang.aspx

接口说明

起点位置gps和终点位置gps，画出导航轨迹【在程序内嵌入浏览器控件,加载接口地址即可】
参数
TravelType:导航方式 driving、bicycling、walking、transit（详情）
startLat:起点纬度
startLng:起点经度
endLat:终点纬度
endLng:终点经度
演示地址
http://api.zdoz.net/daohang.aspx?TravelType=driving&startLat=23.1212&startLng=113.1212&endLat=23.1111&endLng=113.2323

两GPS点之间距离

接口地址

http://api.zdoz.net/getdistance.aspx

接口说明

传入两个gps点的经纬度，返回两个gps的距离，单位米
参数
lat1:第一个gps点的纬度
lng1:第一个gps点的经度
lat2:第二个gps点的纬度
lng2:第二个gps点的经度
演示地址
http://api.zdoz.net/getdistance.aspx?lat1=34.123&lng1=113.234&lat2=34.126&lng2=113.244

接口地址

http://api.zdoz.net/DDD2DMS.aspx

接口说明

传入GPS的度（DDD）表示数据,返回DMS的表示的JSON数据
参数
gps：GPS的DDD表示方法数据
演示地址
http://api.zdoz.net/DDD2DMS.aspx?gps=108.2345

GPS显示方法转换

接口地址

http://api.zdoz.net/DMS2DDD.aspx

接口说明

传入GPS的度（DMS）表示数据,返回DDD表示的数据
参数
D:度
M:分
S:秒
演示地址
http://api.zdoz.net/DMS2DDD.aspx?D=108&M=54&S=22.2

天气预报

接口地址

http://api.zdoz.net/getweather.aspx

http://api.zdoz.net/getweather2.aspx

接口说明

上传城市名根据国家气象局的天其信息查出当天的天气情况
参数
cityName:城市名
参数
lat:GPS纬度
lng:GPS经度

http://api.zdoz.net/getweather.aspx?cityName=天津

http://api.zdoz.net/getweather2.aspx?lat=34.345&lng=113.678
