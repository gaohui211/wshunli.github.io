Hexo使用多说及美化

发表于 2016-08-27   |   分类于 Hexo   |     |   本文阅读量 次

本文主要记录在Hexo、NexT.Mist主题中使用多说的过程。

注意：NexT.Mist主题已经在主题中集成多说，只需在主题配置文件中启用即可。
官方文档请移步：http://theme-next.iissnan.com/third-party-services.html#duoshuo

Hexo 文章集成多说评论

1、下面说一下 Next 主题如何集成多说。首先要在 多说 创建一个站点，具体步骤如下：
2、登录后在首页选择 “我要安装”。
3、创建站点，填写表单。多说域名 这一栏填写的即是你的 duoshuo_shortname，如图：

4、创建站点完成后在 站点配置文件 中新增 duoshuo_shortname 字段，值设置成上一步中的值。

#多说评论
duoshuo_shortname: wangshunli

5、启用后默认在所有页面都会显示多说评论框，比如分类页面、标签页面、自定义的关于页面，如果不想在这些页面显示评论框，找到对应的index.md文档，加入 comments: false 就行了

常规配置

修改多说评论显示UA

在每一条多说评论后显示评论者的信息，需要在 \Hexo\themes\next\_congig.yml中编辑如下字段：

# Make duoshuo show UA
# user_id must NOT be null when admin_enable is true!
# you can visit http://dev.duoshuo.com get duoshuo user id.
duoshuo_info:
  ua_enable: true
  admin_enable: true
  user_id: xxxxxx
  admin_nickname: 博主

自定义CSS美化多说评论框

登陆 多说，点击后台管理，在 设置/基本设置/自定义CSS 中可以修改多说的 CSS 样式，如图：

网友分享的整个多说自定义CSS，其中访客样式和评论样式统一

#ds-reset .ds-avatar img,

embed.js本地化

使用Chrome抓包可以得到多说评论核心脚本embed.js的远程文件地址：

http://static.duoshuo.com/embed.js

在浏览器中打开，右键另存为，放到\theme\next\source\js\src\文件夹中。
再打开\themes\next\layout\_scripts\third-party\comments\duoshuo.swig。
搜索//static.duoshuo.com/embed.js，把它改成/js/src/embed.js。

往你想添加的页面增加下面的一小段代码即可。

<div class="ds-recent-visitors" data-num-items="28" data-avatar-size="42" id="ds-recent-visitors"></div>

其中

class=“ds-recent-visitors”    //指定显示最近访客容器
data-num-items=“28”    //显示最近访客的数量
data-avatar-size=“42”    //显示最近访客头像尺寸大小
id=“ds-recent-visitors”    //为了调节最近访客样式加的
